## Success Criteria

- - User can run /kairos-onboard and complete a semi-structured interview that populates profile (name, timezone, key priorities, habits to track) and confirms vault is ready
- User can run /kairos-north-star and produce populated north-star/desired.md and north-star/dreaded.md files
- User can run /kairos-goals and have scaffolded goal hierarchy files (5-year → yearly → monthly → weekly) in their vault
- Standard Council agent definitions are available in the engine and accessible after setup.sh
- Concierge agent definition exists and is referenced from AGENTS.md
