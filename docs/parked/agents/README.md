|------|-------------|

| `life-coach.md` | Goal and habit coaching advisor |
| `strategic-advisor.md` | Big-picture decision support |
| `accountability-partner.md` | Commitment follow-through |

The live `agents/` directory in the engine repo stays empty (`.gitkeep` only) until a shipped skill explicitly needs a functional agent definition.
