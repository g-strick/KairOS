| Path | Purpose |
|------|---------|
| `inbox/` | Unprocessed captures — `/kair-capture` |
| `daily/` | Daily notes — `/kair-daily` |
| `projects/` | Active projects (one `.md` per outcome) |
| `reference/` | Reference material |
| `next-actions.md` | Actionable now |

| `someday-maybe.md` | Ideas for later |
