# KairOS engine lab (private)

Your private workshop for experiments, drafts, and handoffs. **Never commit this folder to the public engine repo.**

| File | Purpose |
|------|---------|
| `NOW.md` | What you're building this session |
| `BACKLOG.md` | Parked ideas — you pick what becomes NOW |
| `HANDOFF.md` | Update when you stop — resume packet for you or AI |
| `ENGINE-PIN.md` | Dormant public engine: path, branch, commit |

## Folders

- `experiments/` — spikes (one folder per try)
- `drafts/` — skill/script copies before promoting to the public engine
- `handoffs/` — optional dated snapshots of HANDOFF.md
- `archive/` — old planning exports from the engine repo

## Session start

1. Read `HANDOFF.md`
2. Read `NOW.md`
3. Work only on what's in NOW unless you explicitly change it

## Promote to public engine

One file at a time from `drafts/` → engine repo. Run tests there before pushing.
