# Vault inbox

Unprocessed captures land here — from `/kair-capture`.

Run `/kair-triage` to clarify each file (and Reminders Inbox items on macOS) into next actions, projects, waiting-for, someday-maybe, or reference.

This folder is **not** the Reminders "Inbox" list on your phone; that list is consumed by `/kair-triage` via the rem bridge (Reminders are processed first).
